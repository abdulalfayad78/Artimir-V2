# Artimir V2 — Frontend

Interface locale de l’expérience culturelle interactive Artimir, développée
avec React et Vite.

## Commandes

```bash
npm install
npm run dev
npm run lint
npm run build
```

Le frontend doit rester utilisable hors ligne. Les services de mapping, moteur,
caméra et intelligence artificielle seront intégrés séparément.
